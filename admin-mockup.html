CTA Plugin — Shortcode Generator

Generate shortcodes to embed CTA functionality on any Elementor page.

Available Shortcodes

Course Catalog

[cta_course_catalog]

Displays full course grid with filters and pagination.

Single Course

[cta_single_course id="X"]

Displays one course detail page with enroll button and curriculum tabs.

Supervision Booking

[cta_supervision_booking]

Displays Amelia booking calendar for group and individual supervision.

Membership Pricing

[cta_membership_pricing]

Displays pricing cards and bundle comparison table.

Login Form

[cta_login_form]

Displays login and registration form (MemberPress integration).

CE Student Dashboard

[cta_student_dashboard]

Displays CE student dashboard with course progress and certificates.

Supervision Dashboard

[cta_supervision_dashboard]

Displays supervision portal with sessions, documents, and hour tracking.
